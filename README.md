# MediCare AI 🏥

AI-powered Personal Health Assistant with disease prediction, real-time health tracking, and Google Fit integration.

## Features
- Disease Prediction (ML)
- Health Monitoring Dashboard
- Google Fit Integration
- Email OTP Authentication
- Activity & Habit Tracking

## Tech Stack
- Flask (Backend)
- PostgreSQL (Database)
- HTML/CSS/JS (Frontend)
- Machine Learning Models

## Run Locally
pip install -r requirements.txt
python app.py
